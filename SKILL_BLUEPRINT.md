# Алгоритм создания фитнес-плана с нуля

## ВХОД: Анкета клиента
- Возраст, рост, вес, пол
- Травмы/ограничения (грыжи, протрузии, операции)
- Цель (рекомпозиция, похудение, набор массы)
- Частота тренировок (3-4 раза/неделю)
- Место (зал, дом)
- Исключённые продукты (хлеб, сахар, аллергии)
- Бюджет и доступность продуктов

## ВЫХОД: HTML файл
Самодостаточный HTML с base64 картинками упражнений, ~7MB.

---

## ЧАСТЬ 1: ТРЕНИРОВКИ

### Шаг 1. Рассчитать параметры
- BMR по Mifflin-St Jeor: 10×вес + 6.25×рост − 5×возраст ± 5
- TDEE = BMR × коэффициент активности (1.55 для 3-4 тренировок)
- Определить дефицит/профицит по цели

### Шаг 2. Подобрать упражнения под ограничения
**Правила для грыжи L4-L5:**
- ❌ Осевая нагрузка (приседания со штангой, становая тяга, жим стоя)
- ❌ Сгибание поясницы под весом (наклоны, good mornings)
- ❌ Брусья при весе >90кг
- ❌ Болгарские сплит-приседы (нестабильно, нагружает спину)
- ✅ Жим ногами (спина прижата к спинке)
- ✅ Тяги на блоках сидя (без раскачки корпуса)
- ✅ Жимы с опорой спины (скамья 80-85°)
- ✅ McGill Big 3: планка, Dead Bug, Bird-Dog

**Источник картинок:** simplyfitness.com/pages/[exercise-name]
- Скачать PNG, конвертировать в base64, вставить в HTML
- 67 упражнений доступно (см. reference_simplyfitness.md)

### Шаг 3. Структура 4-недельного мезоцикла
- **Неделя 1:** Адаптация (60-70% макс, RPE 6-7)
- **Неделя 2:** Ротация упражнений (65-75%, те же группы — другие движения)
- **Неделя 3:** Объём (верх-ноги-верх, +2.5-5 кг от нед.2)
- **Неделя 4:** Прогрессия (пик интенсивности, RPE 7-8)
- **Неделя 5:** Деload (40% от рабочих весов, 2 подхода, фокус на технику)

### Шаг 4. Формат каждого дня
```
Разминка (10-15 мин): суставная + кардио
Упражнение 1: название, подходы × повторения, отдых, описание техники, ⚠️ предупреждение, альтернативы
...
Упражнение 5-6: ...
Заминка (5-10 мин): растяжка + дыхание
```

Каждое упражнение:
- Ссылка на simplyfitness.com (или другой источник с техникой)
- Картинка base64
- 1-2 альтернативы
- Предупреждение ⚠️ если есть риск для травмы

### Шаг 5. День 4 — Кардио + Кор
- Кардио 20-30 мин (эллипс/велосипед/ходьба с наклоном)
- Кор: планка, Dead Bug, Bird-Dog, вакуум
- Растяжка 10-15 мин

## ЧАСТЬ 2: ПИТАНИЕ

### Шаг 1. Рассчитать КБЖУ
```
Рекомпозиция:
- Тренировочный день: TDEE − 500 ккал
- День отдыха: TDEE − 800 ккал
- Белок: 2 г/кг
- Жиры: 0.8 г/кг
- Углеводы: остаток калорий
```

### Шаг 2. Структура дня
**Тренировочный день (6 приёмов):**
1. Завтрак ~500-600 ккал (одно блюдо! не "омлет + каша отдельно")
2. Перекус ~250-350 ккал
3. Обед ~600-700 ккал
4. До тренировки ~200 ккал (быстрые углеводы: банан, финики)
5. После тренировки ~300-400 ккал (белок + углеводы в первые 30-60 мин)
6. Ужин ~450-550 ккал

**День отдыха (6 приёмов):**
1. Завтрак ~400-500 ккал
2. Перекус ~200 ккал
3. Обед ~550-650 ккал (крупы МЕНЬШЕ чем в тренировочный — 60-70г сухих vs 80-100г)
4. Полдник ~200 ккал
5. Ужин ~450-550 ккал
6. Перед сном ~200 ккал (казеин/творог — медленный белок)

### Шаг 3. Правила составления блюд

**ГЛАВНОЕ ПРАВИЛО: каждый приём пищи = ОДНО реальное блюдо.**

**Реальные завтраки (ПП):**
- Овсяноблин с начинкой (овсянка+яйцо = тесто, начинка внутри)
- Шакшука (яйца в томатном соусе)
- Овсяная каша с яйцом пашот
- Сырники без муки (творог+яйцо+молотая овсянка)
- Творожная миска (творог+орехи+ягоды+овсянка как мюсли сверху)
- Скрэмбл с овощами и мясом

**❌ НЕЛЬЗЯ:**
- "Омлет + овсянка на гарнир" — это два блюда, не одно
- "Яичница + каша отдельно" — не существует как блюдо
- "Протеин смешать с гречкой" — невкусно, есть отдельно
- Фантомные ингредиенты (в рецепте упоминается то, чего нет в составе)
- Дублированное масло (1 ч.л. оливкового масла × 2)

**Правила ингредиентов:**
- Орехи: MAX 15г на порцию, max 2 порции/день
- Масло оливковое: 1 ч.л. (5мл, 40 ккал), НЕ 1 ст.л.
- Крупы: указывать СУХОЙ вес
- Мясо/рыба: указывать ДО термообработки
- Творог: стандартно 5% (не 2% — мало жиров)
- Один источник жиров на приём (не стакать яйца+авокадо+масло+орехи)

**Источники белка (ротация по неделям):**
Курица, индейка, говядина (вырезка), лосось, треска, форель, креветки, кальмары, яйца, творог

**Источники углеводов:**
Гречка, бурый рис, киноа, булгур, овсянка, батат

**Исключено всегда:**
Хлеб, макароны, выпечка, сахар, мёд, соки, газировка

### Шаг 4. Проверка КБЖУ каждого дня
После составления меню — посчитать КБЖУ каждого дня.
Допуск: ±10% от целей.
Если не попадает — скорректировать граммовку (НЕ добавлять новые продукты).

### Шаг 5. Закупки
Для каждой недели сложить ВСЕ ингредиенты из 7 дней.
Категории: Белок, Углеводы, Жиры, Овощи и зелень, Прочее.
Округлить до практичных количеств (1.2 кг, не 1178г).

## ЧАСТЬ 3: РЕКОМЕНДАЦИИ

Добавить после питания:
- Управление весом (взвешивание 3р/неделю, фото каждые 2 недели)
- Бытовая активность (каждые 45-60 мин вставать, поясничная подушка)
- Сон (7-9 ч, 18-20°C, без экранов за 1.5ч)
- Стресс (дыхание 4-7-8, прогулки, пенный ролик)
- План на следующие 4 недели
- Чек-лист ежедневных привычек
- Добавки (изолят, омега-3, магний, витамины, креатин опц.)

## ЧАСТЬ 5: АУДИТ (после генерации)

### 3-фазный аудит
**Фаза 1 (параллельно):** Тренер + Нутрициолог + Юрист
**Фаза 2 (после фазы 1):** Повар читает все 3 аудита
**Фаза 3 (после фазы 2):** QA читает все 4, ищет противоречия

### Известные ложные срабатывания аудитов
- "Пустые закупки" — парсер не извлекает `<table>`
- "Dead Bug/Bird-Dog" — одобрено клиентом
- "Нет ФЗ ссылок" — они в `<div style>` футере, парсер не достаёт

### Критерии приёмки
- Тренер ≥8/10 (безопасность, прогрессия, деload)
- Юрист ≥9/10 (все ФЗ, дисклеймеры)
- Повар ≥8/10 (все блюда реальные, рецепты полные)
- Нутрициолог: средние КБЖУ в ±5% от целей
- Div баланс = 0
- Все ссылки рабочие
